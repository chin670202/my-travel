## 候選景點（每天 4～5 個，圈選後再查票價/時間/車程）

### Day 1 — 08/04（二）彰化
1. 鹿港老街 + 鹿港龍山寺 🍜👨‍👩‍👧 — 古蹟、小吃、伴手禮
2. 台灣玻璃館（黃金隧道）🏠👨‍👩‍👧 — 觀光工廠，鏡屋好拍
3. 彰化扇形車庫 👨‍👩‍👧 — 火車轉盤，小孩愛
4. 八卦山大佛風景區 ⛰️ — 大佛、天空步道
5. 王功漁港（潮間帶/夕陽）🌊👨‍👩‍👧 — 可安排傍晚採蚵/趕海

### Day 2 — 08/05（三）台南
1. 奇美博物館 🏠👨‍👩‍👧 — 必去，西洋藝術/兵器/自然史，超適合全家
2. 十鼓仁糖文創園區 🏠👨‍👩‍👧 — 糖廠改造，擊鼓體驗、溜滑梯
3. 台江四草綠色隧道 🌊👨‍👩‍👧 — 搭船遊紅樹林水道
4. 安平古堡 + 安平老街 🍜 — 古蹟+小吃
5. 頑皮世界野生動物園 👨‍👩‍👧 — 學甲，餵動物互動（路線偏北，順路）

### Day 3 — 08/06（四）高雄
1. 國立科學工藝館 🏠👨‍👩‍👧 — 大型科博館，互動展超多
2. 駁二藝術特區 + 哈瑪星鐵道園區 + 輕軌 👨‍👩‍👧🍜 — 文創、裝置藝術、搭輕軌
3. 旗津（渡輪+海灘+海產）🌊🍜 — 搭渡輪、踩沙、吃海鮮
4. 壽山動物園 👨‍👩‍👧 — 2023 重新開幕，動線好
5. 高雄市立兒童美術館 🏠👨‍👩‍👧 — 適合小孩動手玩

### Day 4 — 08/07（五）高雄
1. 佛光山佛陀紀念館 🏠 — 宏偉、可看展避暑
2. 義大世界（遊樂園 + outlet）👨‍👩‍👧 — 全天玩，水樂園可選
3. 大魯閣草衙道 🏠👨‍👩‍👧 — 商場+小型賽車/摩天輪，雨天備案佳
4. 蓮池潭 龍虎塔 — 地標、傍晚散步
5. 棧貳庫 KW2 + 大港橋 🍜 — 港邊文創、旋轉橋

### Day 5 — 08/08（六）屏東（屏東市/東港一帶）
1. 六堆客家文化園區 🏠👨‍👩‍👧 — 大型園區，親子、室內外皆有
2. 東港華僑市場（海鮮）🍜 — 黑鮪魚生魚片、現流海產
3. 大鵬灣國家風景區 🌊 — 環灣、紅樹林、濕地
4. 屏東縣民公園 — 工業遺構水景，好拍好走
5. 勝利星村創意生活園區 🍜👨‍👩‍👧 — 眷村文創、市集
   > 備案：小琉球（東港搭船約 25 分）海龜浮潛超適合親子，但需多半天～整天，若想去可獨立排一天

### Day 6 — 08/09（日）墾丁（恆春半島）
1. 國立海洋生物博物館 🏠👨‍👩‍👧 — 必去，台灣最大水族館，白鯨/企鵝/海底隧道
2. 後壁湖（浮潛 + 海鮮）🌊🍜 — 餵魚、生魚片
3. 南灣 / 小灣 🌊 — 戲水、玩水上設施
4. 鵝鑾鼻燈塔 + 龍磐公園 — 最南端地標、大草原海景
5. 恆春古城 + 墾丁大街夜市 🍜 — 老街、晚餐逛街

### Day 7 — 08/10（一）台東（走南迴）
1. 多良車站 — 「最美車站」，南迴途中順停
2. 國立台灣史前文化博物館 🏠👨‍👩‍👧 — 2023 整修重開，互動展佳
3. 富山護漁區（潮間帶餵魚）🌊👨‍👩‍👧 — 小孩超愛
4. 加路蘭遊憩區 + 小野柳 — 海岸地景、好拍
5. 台東森林公園（琵琶湖，騎單車）⛰️🌊 — 傍晚騎車涼爽

### Day 8 — 08/11（二）台東
1. 鐵花村 + 台東鐵道藝術村 🍜👨‍👩‍👧 — 慢市集、晚上音樂
2. 知本國家森林遊樂區 ⛰️ — 森林步道、清涼（或知本溫泉泡湯）
3. 原生應用植物園 👨‍👩‍👧🍜 — 火鍋+餵動物，親子友善
4. 三仙台（八拱橋）⛰️ — 海蝕地景（成功鎮，偏北順往花蓮）
5. 池上伯朗大道 + 伯朗咖啡店 👨‍👩‍👧 — 騎單車稻浪（往花蓮方向順路）

### Day 9 — 08/12（三）花蓮（走花東縱谷/海岸）
1. 光復糖廠（花蓮觀光糖廠）🍜👨‍👩‍👧 — 吃冰、日式宿舍
2. 馬太鞍濕地 / 欣綠農園 🌊👨‍👩‍👧 — 生態、捕魚體驗
3. 林田山林業文化園區 🏠 — 小九份，林業聚落
4. 雲山水夢幻湖 — 落羽松、夢幻湖景好拍
5. 鯉魚潭 🌊 — 踩天鵝船、環潭

### Day 10 — 08/13（四）花蓮
1. 遠雄海洋公園 👨‍👩‍👧🌊 — 必選，海洋樂園+表演，全天玩
2. 七星潭 🌊 — 礫石海灣、踏浪看海
3. 立川漁場（摸蜊仔兼洗褲）🌊👨‍👩‍👧🍜 — 下水摸蜆+黃金蜆餐
4. 新光兆豐休閒農場 👨‍👩‍👧 — 餵草泥馬、乳牛、戲水區
5. 東大門夜市 🍜 — 晚餐、原住民美食

### Day 11 — 08/14（五）花蓮 → 宜蘭 → 返家（走蘇花改）
1. 崇德/匯德休憩區 看清水斷崖 — 蘇花途中順停
2. 蘇澳南方澳觀景台 + 內埤海灘 🌊 — 海灣全景
3. 宜蘭傳統藝術中心 🏠👨‍👩‍👧🍜 — 室內外皆有，DIY、表演
4. 蘭陽博物館 🏠 — 單面山建築、宜蘭地理人文
5. 羅東林業文化園區 — 貯木池、森林鐵道（輕鬆收尾）
